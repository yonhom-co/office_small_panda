# 电商行业 Skill

进入电商运营数据分析模式。

## 分析要点
1. **高密度数据结构化**：大促/高并发场景的用户行为、订单、库存、商品数据。
2. **销量与用户洞察**：转化漏斗、流量波动、商品表现，识别影响销量关键因素、流失节点、滞销风险。
3. **交易链路分析**：流量→转化→交易→评价 链路式复盘。

## 推荐分析
- 转化漏斗：浏览→加购→下单→支付 各节点转化率与流失。
- 用户分层：RFM 模型（最近购买/频次/金额）。
- 大促复盘：GMV、客单价、退款率、库存周转。

## 输出规范
- 报告含漏斗图 + 分层散点 + 商品排行柱状图。
- 结论给出运营优化方向（拉新/促活/留存/转化）。
